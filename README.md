SQL Hands-On Practice

This repository contains SQL practice files created to build a strong foundation in writing queries and performing analytical operations on structured data.

Focus Areas

    (a) Basic SQL queries (SELECT, WHERE, ORDER BY)
    
    (b) Filtering, sorting, and aggregations
    
    (c) GROUP BY and HAVING clauses
    
    (d) JOINs across multiple tables
    
    (e) Date functions and time-based analysis
    
    (f) Window functions (ROW_NUMBER, RANK, DENSE_RANK, running totals)

Objective

To strengthen SQL querying and analytical skills, including the use of window functions, through hands-on practice.

Note

This repository is intended for learning and practice purposes, focusing on clarity of logic and practical SQL usage.
